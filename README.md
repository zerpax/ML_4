# Датасеты и метрики качества
## Датасеты
- Классификация: датасет описывает клиентов банка, цель - определить перестал ли пользователь пользоваться банком. Прогнозирование оттока позволяет заранее выявлять пользователей с высоким риском ухода и применять меры удержания.
- Регрессия: датасет описывает дома в Лондоне, цель - предсказать цену дома. Датасет позволяет автоматически оценивать цену на дом, что полезно для покупателей, продавцов, агентств недвижимости и банков при выдаче ипотек и анализе рисков.

## Метрики качества
- Классификация\
    Для оценки общего качества модели будем использовать функцию scikit-learn classification_report(). Она предоставляет метрики precision, recall и f1 для каждого класса. Особенно важным для нас является метрика f1 для класса 1 (пользователи покинувшие банк), так как этот класс является редким в данном датасете, но при этом являетсяглавной целью предсказания модели.
- Регрессия\
    Для оценки качества регрессионной модели будут использованы метрики R2, MAE и MSE . R2 позволяет оценить общее качество модели. MAE показывает среднюю величину ошибки в тех же единицах, что и целевая переменная, что удобно для интерпретации. MSE усиливает влияние крупных ошибок и помогает выявлять модели, у которых сильно отклоняются отдельные предсказания.
